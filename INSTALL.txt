TokenStatus — 설치 방법

1) TokenStatus.app 을 Applications 폴더로 드래그.

2) 첫 실행 (서명/공증이 없어 Gatekeeper가 막으므로 1회만):
   터미널에서:
     xattr -dr com.apple.quarantine /Applications/TokenStatus.app
     open /Applications/TokenStatus.app
   (또는 Finder에서 우클릭 → 열기 → '열기'. 막히면
    시스템 설정 ▸ 개인정보 보호 및 보안 ▸ 아래 '확인 없이 열기')

3) 메뉴바 우측 상단에 사용량이 표시됩니다.

동작 전제 (각자 본인 계정 기준, 안 된 항목은 'N/A'로 뜨고 나머지는 정상):
  - Claude : Claude Code 로그인 (첫 키체인 팝업에서 '항상 허용')
  - GPT    : Codex CLI(codex) 설치 + 로그인  (앱이 codex app-server로 라이브 조회)
  - Gemini : 메뉴바 드롭다운에서 'Antigravity 로그인' 1회 (브라우저 동의)
             ※ agy/antigravity-cli 설치는 불필요 — 앱이 자체 로그인

요구 사항: macOS 14+ / Apple Silicon.
데이터는 각 제공자 공식 API로만 전송되며 제3자로 보내지 않습니다.
